# Генератор идей

Простое консольное приложение на Python, которое генерирует случайные идеи по выбранной категории.

## Что делает программа
- Позволяет выбрать категорию: работа, хобби или технологии
- Выдаёт случайную идею из выбранной категории
- Если категория введена неправильно — выбирает идею из всех категорий
- Можно сохранить понравившуюся идею в файл favorites.txt

## Как запустить
`bash
python main.py
